# 🚀 AI Web Summarizer
A memory-efficient Chrome Extension built for 4GB RAM systems. It offloads AI processing to the Groq Cloud to provide instant article summaries without slowing down your computer.

## 🛠️ Installation
1. **Download** this repository as a ZIP file and extract it.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer Mode** (top right toggle).
4. Click **Load Unpacked** and select the folder you extracted.

## 🔑 Setup
1. Get a free API key from [Groq Console](https://console.groq.com/).
2. Click the **Settings** link in the extension popup.
3. Paste your key and click **Save**.
4. Refresh any webpage and click **Summarize**!
